h1 até h6

São tags usadas para representar titulos e usas hierarquias.

p

É usada para fazer paragrafos.

mark

A tag mark e usada para marcar textos como um marca texto

small

Tag small faz o texto diminuir de tamanho

i

Faz ficar em italico

u

A tag u deixa o texto sublinhado

strong

Deixa em Negrito

ol

Cria uma lista ordenada

ul

Cria uma lista não ordenada

li

São elementos de lista ordenada e não ordenada

a

Cria um link

hr

Cria uma linha horizontal na pagina

sub

Faz o texto ficar um pouco abaixo da linha normal

sup

Faz o texto ficar um pouco acima da linha normal

blockquote

Formata o texto para uma citação

Com grades poderes vem grandes reponsabilidades

font

A tag font foi usada no HTML 4 para especificar a face da fonte, o tamanho da fonte e a cor do texto.

del

Define um texto que foi excluído de um documento traçando uma linha

abbr

Usada para descrever uma abbr
